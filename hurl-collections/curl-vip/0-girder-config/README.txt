To use girder link in inputs (like `magicsGirder:xxxx`), one needs to:
- create an account on girder (https://srmnopt.creatis.insa-lyon.fr/warehouse).
- create a girder API key for VIP (Girder > My Account > API Keys)
- configure it on VIP (see bellow)

List your keys with
../getApiKeys.sh
Edit / Add with necessary
../putApiKey.sh magicsGirder GIRDER-API-KEY
